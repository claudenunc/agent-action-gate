## 1. Program description

The operator runs a personal AI-agent automation system. The system sends SMS messages to the operator's own phone number to deliver status updates, alerts, and approval requests, and to receive the operator's replies.

This program **does not message third parties**. Recipient is the operator's own personal mobile number only. No user opt-in is solicited from anyone other than the operator.

## 2. Message frequency

Up to 30 messages per day, typical operation 5-10 messages per day.

## 3. Message and data rates

Standard message and data rates from the recipient's mobile carrier may apply. The operator (sole recipient) is responsible for any rates charged to the recipient line.

## 4. Help and opt-out

- **HELP** — reply HELP to any message to receive a brief description of the program and contact information.
- **STOP** — reply STOP to any message to immediately discontinue all messaging from this program.
- **START** — reply START after an opt-out to resume messages.

These commands are honored automatically per CTIA guidelines.
